## GitHub Pages

This project is configured to deploy to GitHub Pages automatically using GitHub Actions.

- Push to the `main` branch.
- The workflow will publish the site to the `gh-pages` branch.
- Then enable GitHub Pages in your repository settings and select the `gh-pages` branch.

Your published site URL will be:

`https://ngo-xuantien.github.io/Coffe-Management/`
